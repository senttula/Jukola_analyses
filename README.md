# Jukola_analyses
Analysing the split times of Jukola orienteering relay

Split times can be obtained from https://results.jukola.com/tulokset/en/ as xml file.
Each year has 1000-1600 7 person mens teams and similar amount of 4 person womens teams.
Each person has 10-30 controls which gives us millions of datapoints to analyse from.
